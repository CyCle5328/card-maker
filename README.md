# card-maker 프로젝트

firebase를 이용하여 로그인 및 실시간 데이터베이스를 구축하여 해당 아이디로 저장된 buisness card를 생성, 수정, 제거할 수 있는 프로젝트입니다.

## 사용된 기술스택
+ React
+ JavaScript, JSX
+ HTML
+ PostCSS
